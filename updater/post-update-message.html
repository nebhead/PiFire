PiFire 1.11.0 Release - TBD 2026

  WARNING: This is a development release. Use this at your own risk. There are known issues with this release.

This release contains bug fixes and feature updates since the previous version.


  Here are just some of the NEW features that were added in this update:

  • Default Dashboard: Default dashboard now shows dashes on the gauges when stopped to help avoid confusion that any temperatures are being actively displayed when stopped.
  • Flex Display Improvements: Updates the Flex display to show dashes on the gauges when stopped. Hopefully this will help avoid confusion that any temperatures are being actively displayed when stopped.
  • ETA Calculation Disable: You can now disable estimated time to target calculations for probes on the dashboard via a new setting in the Global Dashboard Settings. This is useful for users who do not want ETA calculations to be performed.
  • Settings Page Improvements: Organized into sections that make more sense (Frontend Settings, Operations, Probe Settings). Work Mode Settings is now Controller Settings to better reflect its purpose.
  • Behind the scenes improvements including:
    • Changed PCB 4.x.x defaults for relay trigger levels (active HIGH by default) to match the v1.x.x relay boards behavior.

  Here are some bug fixes in this update:

  • Pellet Manager: Fixed missing pellet manager link in the navigation menu.
  • Navigation Menu Highlighting: Fixed issue where the navigation menu did not highlight the current page correctly in some cases.
  • ILI9488 Wizard Typo: Fixed Typo in ILI9488 Wizard Manifest Information.
